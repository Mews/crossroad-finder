# crossroad-finder

A website for finding specific nether fortress crossroad patterns

Built on top of my [crossroad-finder api](https://github.com/Mews/crossroad-finder-api), which uses the [CrossroadFinder tool](https://github.com/Gaider10/CrossroadFinder) by [Gaider10](https://github.com/Gaider10)

Visit the website at [https://mews.github.io/crossroad-finder/](https://mews.github.io/crossroad-finder/)
